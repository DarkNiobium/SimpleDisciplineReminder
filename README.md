# SimpleDisciplineReminder
Simple background python app for Windows 10.
Shows toast-notification "WORK." in every 25 minutes and lives in tray.

If you want to use it on every reboot i recommend to it to `shell:startup`.

## Installation:
Install requirements:
  `pip install pystray pillow win10toast`
